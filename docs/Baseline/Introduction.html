This section of the website explores the baseline data for the PREdiCCt study, generating inferences and producing display items for publication.

For all baseline reports, only the faecal calprotecin cohort, the sub-cohort defined as subjects with baseline faecal calprotecin available, are described (unless otherwise stated). For each variable considered, associations with faecal calprotectin categories (<50, 50–250, >250) are presented using chi-squared tests and ANOVA for categorical and continuous data respectively.

There are three main data sources for baseline demographic and phenotypic data in PREdiCCt:

  1. Participant questionnaires.
  2. Forms completed by a subject’s clinical team (via REDCap) during recruitment.
  3. Data collected retrospectively during the end-of-study (EoS) phenotyping which was completed by a member of the subject’s IBD care team.

As all three are used, it will be made clear which source the data presented originate from.
